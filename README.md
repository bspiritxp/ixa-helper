<a id="installing-yarn"></a>

## Install Yarn

On MacOS you can run `=brew install yarn --ignore-dependencies=` to install Yarn, otherwise you can download an installer from [here](http://yarnpkg.com/en/docs/install).
Install dependencies

    yarn install

or

    yarn
